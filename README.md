# ⚽ ЧМ 2026 — живой трекер (версия под Vercel)

Живой сайт с автообновлением результатов из football-data.org.

## Что внутри
- `index.html` — весь фронтенд (расписание 104 матчей, группы, сетка плей-офф, фильтры).
- `api/matches.js` — серверная функция Vercel (прокси к API, прячет ключ).

Если API недоступен — сайт автоматически показывает зашитое расписание и сыгранные результаты, так что страница работает всегда.

## Деплой за 4 шага
1. **API-ключ.** Зарегистрируйся на https://www.football-data.org/client/register и получи API Token (32 символа).
2. **GitHub.** Создай репозиторий `wc2026` (Public) и загрузи в него `index.html` и папку `api/`.
3. **Vercel.** vercel.com → Add New Project → выбери репозиторий. Framework Preset: **Other**, Build Command и Output Directory оставь пустыми.
4. **Ключ в окружение.** Settings → Environment Variables → добавь `FOOTBALL_API_KEY` = твой ключ. Нажми **Deploy**.

Через ~1 минуту получишь ссылку вида `wc2026-xxxxxx.vercel.app`.

## Обновление расписания вручную
Открой `index.html`, найди `const WC_DATA = {...}` в начале `<script>`, поправь нужные матчи и снова закоммить в GitHub — Vercel пересоберёт сайт автоматически.

> Примечание: бесплатный план football-data.org — 10 запросов/мин. Серверная функция кэширует ответ на 30с, чтобы не упереться в лимит.
